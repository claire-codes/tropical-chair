# Angular 1.x Seed App

The most basic of basic.

## Setup

```
npm install
```

* Run app with `npm start`
* Run Grunt tasks (automatic Sass compilation) with `grunt`
* Run tests with `karma start`

### Does have
* Components
  * Controller
  * Service - factory
* Tests
  * Unit test Controller
  * Unit test Service - factory
* Sass compilation
* Run on node

### Doesn't have _yet_
* Directive
* Unit test directive
